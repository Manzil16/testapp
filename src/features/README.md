# Feature Modules

This directory is organized by business capability:

- `auth`: authentication/session and route protection
- `users`: user profiles and role metadata
- `vehicles`: vehicle profiles and range defaults
- `chargers`: charger discovery, details, and host charger CRUD
- `bookings`: booking request and lifecycle state
- `reviews`: post-session rating and trust signals
- `notifications`: in-app notification records
- `verification`: admin verification queue and moderation notes
- `trips`: trip planning and route overview data
- `shared`: shared theme, utilities, and cross-feature types
